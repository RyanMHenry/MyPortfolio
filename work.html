Summary

This game is built to resemble the classic title Star Fox 64. It contains a rail system that controls the game, 3 enemy classes with different patterns of attack and movement. It also contains 4 different power ups that all shoot differently, some with different bullets, others with different patterns. It also has a simple prototype save system using only system.IO for reference.
